# Lab 2 - Calcul d'Impôt

Dans ce lab, j'ai créé une application Android en Java qui calcule 
l'impôt d'une maison. L'utilisateur entre la surface en m², le nombre 
de pièces, et coche s'il y a une piscine. En cliquant sur le bouton 
"Calculer l'impôt", l'application affiche le montant total à payer.
